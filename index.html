General information about the *.chm format

HTMLHelp (Microsoft Compressed HTML Help, Microsoft Compiled HTML Help,. CHM) - a proprietary file format for context-sensitive help, developed by Microsoft and released in 1997 as a replacement format WinHelp. Contains a set of HTML-pages can also include links to content pages, index, as well as a base for full-text search on content pages. All members of the. CHM files are compressed by the algorithm LZX.

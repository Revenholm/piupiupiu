Проявите свои творческие способности. После изучения основ цифровой фотографии вам захочется воспользоваться и дополнительными возможностями. В состав Windows XP входит приложение Paint, позволяющее легко обрезать цифровые фотографии, изменять их размеры и ретушировать. Впрочем, творить удобнее с помощью специального программного обеспечения для редактирования изображений с гораздо более широким набором возможностей. Программы наподобие Microsoft Picture It! позволяют улучшать фотографии, а затем делать из них открытки, календари и т. п. Для ознакомления с другими идеями посетите сообщество Picture It! в MSN.

Кроме того, существуют профессиональные программы, такие как Microsoft Publisher, позволяющие создавать рекламные материалы и другие публикации с использованием собственных фотографий.

Возможности поистине безграничны!
